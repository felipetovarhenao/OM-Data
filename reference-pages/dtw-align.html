DTW-ALIGN

[GENERIC-FUNCTION]
OM-Data
Function Reference
ARGUMENTS:
  - a list [default = (0 1 2 3 4 2)]
  - b list [default = (0 2 3 4 1)]
  - posn list [default = ((0 0) (1 0) (2 1) (3 2) (4 3) (5 4))]

Description:

Works in combination with DTW, aligning the values of the main time series to the others. Use the left input of DTW as in1, and the outputs of DTW as in2 and in3, respectively.

Example:
(dtw-align '(0 1 2 3 4 2) '(0 2 3 4 1) '((0 0) (1 0) (2 1) (3 2) (4 3) (5 4))) => ((0 0) (1 0) (2 2) (3 3) (4 4) (2 1))
