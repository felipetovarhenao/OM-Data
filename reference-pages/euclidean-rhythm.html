ARGUMENTS:
  - numbeats integer [default = 5]
  - period integer [default = 13]
  - rotation integer [default = 0]

Description:

euclidean-rhythm, as the name suggests, computes the Euclidean rhythm for a given period k and number of beats/pulses n, where n < k. The output can be interpreted as a list of intervals or distances between between pulses
