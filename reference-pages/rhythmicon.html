RHYTHMICON

[GENERIC-FUNCTION]
OM-Data
Function Reference
ARGUMENTS:
  - base-dur number [default = 3000]
  - subdivisions list [default = (quote (1 2 3 4 5 6 7))]
  - times integer [default = 4]

Description:

Outputs a rhythmicon as a MULTI-SEQ, given a fundamental duration (ms), a list of subdivisions, and a number of repetitions. For each rhythmic partial, the corresponding pitch is automatically assigned, using 3300 as the fundamental.
